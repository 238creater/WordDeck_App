# Word Deck

ブラウザだけで動く単語帳アプリです。市販単語帳のデータはリポジトリに含めず、利用者が自分のCSVをブラウザで取り込む想定です。

## 使い方

`index.html` をブラウザで開きます。

CSVの列名は次の形式です。

```csv
lesson,english,japanese
Stage1-動詞,run,走る/経営する/実行する
Stage1-動詞,make,作る/〜させる
Stage1-名詞,water,水
多義語,right,右/正しい/権利
```

取り込んだ単語帳はブラウザのLocalStorageに保存されます。

`lesson` に数字を入れると `Lesson 1` のように表示されます。`多義語` など文字を入れると、Lessonとは別枠の学習範囲としてそのまま表示されます。

学習設定では出題範囲を複数選択できます。`すべて` を押すと全範囲の選択/解除を切り替えます。何も選んでいない状態で開始する場合は、全範囲から出題します。

`Stage1-動詞` のように `-` で区切ると、学習設定では `Stage1` の折りたたみ内に `動詞` として表示されます。`-` がない `多義語` などは `その他` にまとまります。

折りたたみ内の品詞は `動詞`、`名詞`、`形容詞`、`副詞・その他` の順に並びます。CSVに `副詞` や `その他` と書いた場合も、表示上は `副詞・その他` として扱います。

TOEIC単語帳を単語数で細分化する場合は、たとえば `Stage1-001-100`、`Stage1-101-200` のように書くと、`Stage1` の折りたたみ内に範囲別でまとまります。

## 学習モード

- 日本語 → 英語
- 4択 英語 → 日本語
- 4択 日本語 → 英語

出題数は 10問、30問、50問、100問、全部、エンドレスから選べます。

通常の出題数では、選択範囲の単語数が足りない場合は開始できません。`全部` は選択範囲の全単語をランダムに1回ずつ出題します。`エンドレス` は選択範囲をランダムに一周し、全部出したら再シャッフルして繰り返します。

PCでは回答後に Enter で次へ進めます。4択では 1〜4 キーでも選択できます。
