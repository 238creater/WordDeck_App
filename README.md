# Word Deck

ブラウザだけで動く単語帳アプリです。市販単語帳のデータはリポジトリに含めず、利用者が自分のCSVをブラウザで取り込む想定です。

## 使い方

`index.html` をブラウザで開きます。

CSVの列名は次の形式です。

```csv
lesson,english,japanese
Stage1-動詞,run,走る/経営する/実行する
Stage1-動詞,make,作る/〜させる
Stage1-名詞,water,水
多義語,right,右/正しい/権利
```

TOEICなどの文法・例文穴埋め用CSVも取り込めます。

```csv
stage,question_english,question_japanese,choices,answer,explanation
Stage1-接続詞,Jackets are required for club members in the main dining room _ casual dress is acceptable in all other areas of the club.,クラブの他の場所では普段着が許される一方、主食堂内では、クラブ会員はジャケットが必要である。,while/during/because/unless,while,while は「一方で」を表し、前後の内容を対比します。
Stage1-接続詞,The meeting was canceled _ the heavy rain.,大雨のため会議は中止された。,because of/despite/although/while,because of,because of の後ろには名詞句を置きます。
```

穴埋め形式では `question_english` に `_` を1つだけ入れます。`choices` は `/` 区切りで4つ、`answer` はその中の正解を書きます。`explanation` は任意で、短い解説を書いておくと回答後に表示できます。

取り込んだ単語帳はブラウザのLocalStorageに保存されます。

学習中に問題カード右上の `しおり` を押すと、その単語をしおり単語として保存できます。学習設定ではしおり単語の一覧確認、解除、しおり単語だけでの出題ができます。しおり単語の情報もCSVではなくブラウザのLocalStorageに保存されます。

`lesson` に数字を入れると `Lesson 1` のように表示されます。`多義語` など文字を入れると、Lessonとは別枠の学習範囲としてそのまま表示されます。

学習設定では `範囲を変更` から出題範囲を複数選択できます。`すべて選択` / `すべて解除` で全範囲の選択を切り替えます。何も選んでいない状態では開始できません。

`Stage1-動詞` のように `-` で区切ると、学習設定では出題範囲一覧に `Stage1` のまとまりが作られ、その中に `動詞` として表示されます。`-` がない `多義語` などは `その他` にまとまります。

品詞は `動詞`、`名詞`、`形容詞`、`副詞・その他` の順に並びます。CSVに `副詞` や `その他` と書いた場合も、表示上は `副詞・その他` として扱います。

TOEIC単語帳を単語数で細分化する場合は、たとえば `Stage1-001-100`、`Stage1-101-200` のように書くと、`Stage1` の中に範囲別でまとまります。

## 学習モード

単語帳形式:

- 日本語 → 英語
- 4択 英語 → 日本語
- 4択 日本語 → 英語

穴埋め形式:

- 記述 穴埋め
- 4択 穴埋め

出題数は 10問、30問、50問、100問、全部、エンドレスから選べます。

通常の出題数では、選択範囲の単語数が足りない場合は開始できません。`全部` は選択範囲の全単語をランダムに1回ずつ出題します。`エンドレス` は選択範囲をランダムに一周し、全部出したら再シャッフルして繰り返します。

4択モードでは、選択範囲に4語以上あり、選択肢として表示できる異なる答えが4つ以上ある場合だけ開始できます。

`チャレンジモード` をオンにすると画面テーマが切り替わり、1問でも間違えた時点で結果画面に移動します。チャレンジモード中はエンドレスを選べません。

PCでは回答後に Enter で次へ進めます。4択では 1〜4 キーでも選択できます。
